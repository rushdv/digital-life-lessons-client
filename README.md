# Digital Life Lessons

**Live Site:** [https://your-live-site-url.web.app](https://your-live-site-url.web.app)

## About the Project

Digital Life Lessons is a full-stack platform where users can create, store, and share meaningful life lessons, personal growth insights, and wisdom gathered over time.

## Features

- 🔐 Secure authentication with Email/Password and Google Sign-In via Firebase
- 💎 Free & Premium subscription plans with Stripe payment integration (one-time lifetime access)
- 📖 Browse public life lessons with search, filter by category/emotional tone, and sort options
- 🔒 Premium lessons are blurred/locked for free users with an upgrade prompt
- 📊 User and Admin dashboards with analytics charts, lesson management, and user management
- 🔖 Save lessons to personal Favorites and manage them from the dashboard
- ❤️ Like, comment, report, and share lessons with social media integration
- 🛡️ Admin panel to manage users, lessons, featured content, and reported/flagged lessons
- 📱 Fully responsive design for mobile, tablet, and desktop
- 🎞️ Lottie animations for enhanced user experience

## Tech Stack

**Client:** React, Vite, Tailwind CSS, React Router DOM, TanStack Query, Firebase, Axios, Recharts, Swiper, React Hook Form, React Hot Toast, SweetAlert2, Lottie React, React Share

**Server:** Node.js, Express.js, MongoDB, JWT, Stripe, Firebase Admin SDK
